# Portafolio-Analisis-de-Sistemas
Este repositorio contiene el trabajo realizado para la materia Análisis de Sistemas, perteneciente a la carrera Ingeniería en Informática de la Universidad Nacional de La Matanza (UNLaM).

El portafolio reúne técnicas como Modelo de Dominio, Análisis Orientado a Objetos, UML, Casos de Uso, Diagramas de Actividades, Diagramas de Estado, Modelos de Análisis, entre otros.
